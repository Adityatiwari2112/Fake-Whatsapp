# Fake WhatsApp 💬

A simple WhatsApp-inspired chat application built using JavaScript. This project simulates the basic functionality and user interface of a messaging app, allowing users to send and display messages in a chat-like environment.

## 🚀 Features

* Send messages instantly
* Dynamic chat interface
* User-friendly design
* Responsive layout
* Auto-update chat window
* Simple and lightweight implementation

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

```bash
Fake-WhatsApp/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 📸 Preview

The application includes:
